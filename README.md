# DAA Algorithms in C

This repository contains implementations of various **Data Structures and Algorithms (DAA)** problems in **C language**. Each code file implements a specific algorithm or problem using different approaches like recursion, divide and conquer, dynamic programming, etc.

## Array and Permutation
- **array_permutation.c** → Generates all possible permutations of an array using recursion.

---

## Big Integer Operations
- **bigint_linkedList.c** → Implements addition, multiplication, and factorial of big integers using linked lists.

---

## Sorting Algorithms
- **counting_sort.c** → Counting sort for integers in a given range.  
- **radix_sort.c** → Radix sort using counting sort for each digit.  
- **merge_sort.c** → Standard divide-and-conquer merge sort.  
- **hybrid_merge_sort.c** → Hybrid merge sort using insertion sort for small subarrays.  
- **quicksort_fixed_random.c** → Quick sort with fixed and randomized pivot options.

---

## Searching Algorithms
- **binary_search_recurssive.c** → Standard recursive binary search.  
- **ternary_search_recursive.c** → Recursive ternary search.

---

## Recursion Problems
- **fibonacci_method.c** → Fibonacci numbers using naive recursion, memoization, and iterative DP.  
- **pascal_triangle_method.c** → Pascal’s triangle using naive recursion, memoization, and iterative DP.  
- **power_divide_conquer.c** → Computes \(a^n\) using divide and conquer.  
- **max_element_recurssion.c** → Finds the maximum element in an array using recursion.  
- **min_max_divide_conquer.c** → Finds minimum and maximum elements with divide-and-conquer.

---

## Dynamic Programming
- **treasure_island_dp.c** → Maximum treasure collection problem (DP).  

---

## Matrix Problems
- **matrix_determinant.c** → Computes determinant of a matrix using recursion.  
- **matrix_multiplication_dac.c** → Multiplies two matrices using divide-and-conquer.  
- **walsh_matrix.c** → Generates Walsh matrix of a given order.

---

## Set Operations
- **set_operation.c** → Implements union, intersection, difference, complement, subset, Cartesian product, and power set for sets.

---

### Usage

1. Open each `.c` file in a C compiler (like `gcc` or VS Code with C extensions).  
2. Compile and run to see the algorithm in action.
